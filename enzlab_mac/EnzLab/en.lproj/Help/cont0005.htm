Guide to EnzLab menus

The following are the menu commands in EnzLab.

File

  • Save: Saves your current work
  • Load: Reloads work from disk
  • Restart: Restarts EnzLab
  • Printer Setup: Allows you to change the printer setup
  • Exit: Ends EnzLab

Enzyme

  • Information: Displays information about your enzyme, including its type, reference number etc.
  • Reselect: Allows you to reselect your enzyme. If you have not yet selected an enzyme, this menu item is listed as Select. N.B. This Restarts EnzLab if you are reselecting your enzyme.

Results

  • List experimental results: Invokes the List of Experimental Results dialog box, which lists the experiments you've so far performed.
  • List kinetic analyses: Invokes the List of Kinetic Analyses dialog box, which lists the kinetic analyses you've so far performed.
  • Copy experimental results to clipboard: puts the results of your experiments in the Windows Clipboard so you can paste them into another Windows application (e.g. spreadsheet, graphics package). See Copying experimental results to Clipboard for more information.

Analysis

  • Precision: Allows you to assess experimental precision.
  • Kinetics: Allows you to perform a kinetic analysis.
  • pH Profile: Allows you to obtain a pH profile.

Go Home

  • Go Home: Ends the current simulation day.

Help

  • Contents: Displays the contents page of EnzLab Help.
  • What to do next: Makes a suggestion about what you should do next.
  • About EnzLab: Displays a little information about the program.
