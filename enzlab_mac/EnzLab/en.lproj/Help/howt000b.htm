Reloading work from disk

You must have a previously saved file on disk holding the data you wish to reload.

Choose Load from the File menu. The Load dialog box will appear - enter the name of the file you saved and click on OK.

A message will appear telling you if the program successfully loaded the data from disk.

If the program found the file, all the experimental data from your previous session, the simulation time and the amount of enzyme you had left will be reloaded into the program's memory.
