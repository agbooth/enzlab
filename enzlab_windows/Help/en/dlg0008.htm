New Day dialog box

The New Day dialog box is where you choose the number of frozen samples of enzyme to thaw.

No. to thaw
Type the number of samples of enzyme you wish to thaw in the box.

OK button
The OK button stays grayed until you have chosen a valid number of samples - if you don't want to thaw any, put '0'.
When you have typed the number to thaw, click on OK to thaw the samples.
