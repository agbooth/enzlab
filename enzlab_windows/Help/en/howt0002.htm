$Using pull-down menus

#KMicrosoft Windows uses pull-down menus.

To use a command from a menu, you must first select the menu, and then choose the command you want.

Selecting and opening a menu
You can select a menu in any of the following ways:

·Using the mouse, point to the name of the menu on the menu bar and click the left mouse button.

·Press Alt or F10 to highlight the menu bar, then use the left and right arrow keys to select the menu you want. Press Enter to open the selected menu.

·If a name in the menu bar has an underlined letter, press Alt to highlight the menu bar, then the underlined letter.

Choosing a menu command
You can choose an item from an open menu in any of the following ways:

·Click the item with the mouse.

·Type press the letter that is underlined in the item name.

·Move the highlight with the up and down arrow keys until the item you want is highlighted, then press Enter.

Closing a menu
To close a menu you have opened you can:

·Click the menu name or anywhere outside the menu.

·Press Alt or F10 to cancel the menu.

·Press Esc to close the menu but remain on the menu bar so you can select another menu.
