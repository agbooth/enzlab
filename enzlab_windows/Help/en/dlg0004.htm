Examination of Kinetic Analysis Results dialog box

The Examination of Kinetic Analysis Results dialog box shows which experimental points were taken at substrate concentrations greater than 50 times the value you have just obtained for Km.

These points tell you very little about your enzyme. Consequently, the program will label these points as 'N/A', for 'Not Available', from now on. You will not be able to use these points in any subsequent analyses.

The graph of your results is shown, with a dotted vertical line marking 50 Km. Any points to the right of this line will be marked as 'N/A'.

OK button
Click on OK when you have finished.
