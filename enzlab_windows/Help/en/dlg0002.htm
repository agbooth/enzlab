Enzyme Selection dialog box

The Enzyme Selection dialog box is where you select the type of enzyme with which you wish to work.

Enzyme Type
Choose the enzyme type you want by clicking on the appropriate name.

OK button
Click on OK when the correct enzyme is selected.

Cancel button
Clicking on Cancel will remove this dialog box without selecting an enzyme. You won't be able to perform any experiments until you have selected an enzyme. (You can do this by choosing Select from the Enzyme menu.) You will, however, be able to reload previously saved experimental data, by choosing Load from the File menu.
