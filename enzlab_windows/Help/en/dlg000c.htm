Precision Analysis Selection dialog box

The Precision Analysis Selection dialog box is where you select the experiments for an analysis of experimental precision.

List box
Experiments that are currently selected are highlighted in the list box.

Selecting and Unselecting experiments

  • In the boxes to the left, type the values of the parameters (Vol, pH, S and I) of the experiments you want.
    All experiments whose parameters match those in the boxes are selected.
  • Click with the mouse on an unselected experiment in the list box.
    All experiments whose parameters match that experiment's are selected, and those parameters put into the boxes.
  • Hold down Ctrl on the keyboard while clicking on an already selected experiment.
    That experiment (alone) will be unselected.
  • Hold down Ctrl on the keyboard while clicking and dragging over a range of already selected experiments.
    Those experiments will be unselected.

Initially, the boxes will hold the values from the last experiment you performed, and all matching experiments will be highlighted.

No. of expts selected
The box at the bottom of the screen keeps count of how many experiments you have selected - you need at least five.

OK button
When you have selected the experiments you wish, click on OK to perform the analysis.

Cancel button
If you don't want to perform a precision analysis, click on Cancel.
