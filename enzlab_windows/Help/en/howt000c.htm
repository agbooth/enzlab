Assessing experimental precision

Experimental data required

You need to perform the same experiment at least five times (i.e. using the same volume, pH and concentration of substrate and inhibitor). [ Unsuccessful experiments don't count! ]

Analysing the data

Choose Precision from the Analysis menu. This brings up the Precision Analysis selection dialog box. Here you select the experiments you want to include in the precision analysis; remember that they all must have the same parameters.

Once you have selected, the program displays the experimental results you selected, along with the mean and standard deviation of the rates measured (values of Vo).
